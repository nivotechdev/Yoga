# **App Name**: Equilibrium Yoga Space

## Core Features:

- Informational Content: Displays static content from the original website, adapted for a Yoga Space including home, services, about, testimonials, and contact pages.
- Class Scheduling: Allows users to view the class schedule and book classes.
- Contact Form: Allows users to send a message via a contact form. After submission the contact data is sent to the Yoga space email address.

## Style Guidelines:

- Primary color: Soft sage green (#A8BFA3) for a natural and calming feel.
- Background color: Light sand beige (#EDE6D6) for a clean, sophisticated look.
- Accent color: Deep earthy brown (#5C4A3D) to ground the design.
- Font pairing: 'Alegreya' (serif) for titles and headings, paired with 'PT Sans' (sans-serif) for body text, provides a balance of elegance and readability.
- Use simple, line-based icons to represent yoga poses and wellness concepts.
- Implement fade-in animations on scroll to reveal content smoothly.
- Subtle parallax effect in the hero section to add depth.
- Subtle hover animation on buttons to provide feedback on user interaction
- Use smooth transitions when changing between sections of the web app.
